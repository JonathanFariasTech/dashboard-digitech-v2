# 📊 Dashboard Digitech v2.0

Sistema de gestão educacional completo baseado em **Streamlit** + **Supabase** + **GitHub**.

## 🎯 Funcionalidades

- ✅ Upload e importação de planilhas Excel mensais
- ✅ Armazenamento persistente no banco de dados PostgreSQL (Supabase)
- ✅ Visualização de KPIs e métricas educacionais
- ✅ Análise de Hora-Aluno com metas configuráveis
- ✅ Controle de docentes, turmas, alunos e ambientes
- ✅ Histórico temporal e tendências
- ✅ Exportação de relatórios em CSV
- ✅ Auditoria de todas as operações
- ✅ Deploy automatizado no Streamlit Cloud
